Альтернатива

Объединять PDF из контекстного меню

macOS Quick Action: правый клик по файлам → “Merge PDFs”. Порядок не берётся как есть — делается числовая сортировка по имени (1,2,3…). Качество сохраняется: страницы копируются.

Как встроить на Mac

1. Установи qpdf

Открой Терминал и выполни:

$ brew install qpdf

2. Скачай .workflow

Файл появится в “Загрузки”. Дважды кликни по нему и нажми “Установить”. Готово.

Merge PDFs.workflow
Скачать .workflow 
# Merge PDFs (workflow):
# 1) берет выбранные PDF из контекстного меню
# 2) числовая сортировка по первому числу в имени (stable: если чисел нет — порядок не ломается)
# 3) объединяет через qpdf (страницы копируются, без рендеринга)
#
# out: ~/Desktop/Merged-PDF/<base>-merged.pdf
#
# Сортировка (идея):
#   python3 - <<'PY' "$@"  → печатает пути в нужном порядке
#   mapfile -t sorted < <(python3 - ... )
#
# Склейка (идея):
#   qpdf --empty --pages "${sorted[@]}" -- "$out"

3. Как пользоваться

В Finder выдели несколько PDF → правый клик → “Quick Actions” → “Merge PDFs”. Готовый файл появится на Desktop в папке Merged-PDF/.

Как встроить в контекстное меню Windows

1. Установи одной командой

Открой PowerShell от имени администратора и выполни:

windows/install-merge-pdf.ps1 
$base='https://raw.githubusercontent.com/rafael-mansurov/remove-watermark/main/windows'
$ErrorActionPreference = 'Stop'
$installDir = Join-Path $env:LOCALAPPDATA 'MergePDF-ContextMenu'
New-Item -Path $installDir -ItemType Directory -Force | Out-Null
$root = $base.TrimEnd('/')

irm "$root/install-merge-pdf.ps1" | iex

# Добавит пункт “Merge PDFs” в контекстное меню для PDF.

2. Как пользоваться

Выдели PDF → правый клик → “Merge PDFs”. Скрипт сделает числовую сортировку по имени и объединит всё через qpdf.

3. Как удалить

Выполни в PowerShell:

$ powershell -NoProfile -ExecutionPolicy Bypass -File "$env:LOCALAPPDATA\\MergePDF-ContextMenu\\uninstall-merge-pdf.ps1"
Вопросы

Частые вопросы

Качество реально не портится?

Да. Склейка делается копированием страниц PDF. Мы не конвертируем страницы в картинки и не пересобираем их заново — поэтому визуальные данные и векторная графика сохраняются.

Куда уходят файлы?

Никуда. Склейка работает локально в браузере. На сервер отправляются только скрипты из CDN.

Можно ли объединять очень много PDF?

Технически — да, но упирается в память браузера. Лучше пробовать частями, если у тебя тысячи файлов или очень большие PDF.
