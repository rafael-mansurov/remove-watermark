Набор инструментов

Работа с файлами
прямо в браузере

Инструменты для работы с файлами прямо в браузере. Без регистрации. Файлы остаются у тебя на устройстве

Фото

Убрать водяной знак

Загружаешь фото, выделяешь зону с водяным знаком и браузер восстанавливает фон под ним. Скачиваешь чистое изображение

  • Работает офлайн
  • HEIC, PNG, JPG, WEBP
  • Бесплатно в браузере
PDF

Объединить PDF

Перетаскиваешь несколько PDF, расставляешь нужный порядок и скачиваешь один готовый документ

  • Любое количество файлов
  • Настройка степени сжатия
Фото

Сжать в WebP

Размер падает в 3–5 раз, а качество остаётся таким же. Выбираешь степень сжатия сам. Поддерживает пакетную обработку

  • Пакетная обработка
  • HEIC, JPG, PNG → WebP
  • Слайдер качества
