Набор инструментов

Работаем с файлами
прямо в браузере

Три инструмента для работы с файлами. Без установки. Без регистрации. Файлы остаются у вас.

Фото

Убрать водяной знак

Загружаешь фото, выделяешь зону с водяным знаком — браузер восстанавливает фон под ним. Скачиваешь чистое изображение.

  • Работает офлайн
  • HEIC, PNG, JPG, WEBP
  • Бесплатно в браузере
PDF

Объединить PDF

Перетаскиваешь несколько PDF, расставляешь нужный порядок — скачиваешь один готовый документ.

  • Любое количество файлов
  • Порядок страниц — руками
  • Настройка сжатия
Фото

Сжать в WebP

Размер падает в 3–5 раз — качество остаётся. Выбираешь степень сжатия сам. Поддерживает пакетную обработку.

  • Пакетная обработка
  • HEIC, JPG, PNG → WebP
  • Слайдер качества
