Python · macOS · OpenCV

remove_watermark.py

Удаляет белый текст и водяные знаки с любых фото. Интерактивный редактор зоны перед каждым снимком — тяни за углы, смотри результат сразу. Работает полностью локально, без интернета.

01 / Установка

Зависимости

Открой Терминал, вставь команду и нажми ENTER.

$ brew install python && pip3 install opencv-python numpy
02 / Скачать скрипт

remove_watermark.py

Нажми «Скачать .py» и файл сохранится в «Загрузки».

remove_watermark.py
Скачать .py 
Загрузка скрипта…
03 / Запуск

Как запустить

После скачивания просто открой Терминал на macOS, вставь команду ниже и нажми ENTER — она сама перейдёт в папку «Загрузки» и запустит скрипт. В появившемся окне выбери папку с фото (не отдельный файл).

$ cd ~/Downloads && python3 remove_watermark.py

Обработанные фото сохраняются в папку cleaned/ внутри выбранной папки. Уже обработанные файлы пропускаются автоматически.

04 / Редактор

Управление

Перед каждым фото открывается окно. Слева — оригинал с выделением зоны, справа — результат в реальном времени.

Drag: углы
Изменить размер зоны удаления
Drag: середина
Переместить зону целиком
Threshold
Порог яркости — выше = только самые белые пиксели
Radius
Радиус inpainting (рекомендуется 2–5)
ENTER / SPACE
Обработать это фото и перейти к следующему
S / N
Пропустить фото без обработки
ESC
Выйти из программы
